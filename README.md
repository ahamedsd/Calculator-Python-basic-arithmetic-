Simple Calculator in Python

This is a beginner-friendly Python project that demonstrates a basic calculator capable of performing addition, subtraction, multiplication, and division. It is designed for learners who are just starting with Python programming.

Features:

Perform basic arithmetic operations: Add, Subtract, Multiply, Divide

Handles division by zero

Interactive menu-driven interface

Easy to read and understand for beginners

How to Run:

Make sure Python is installed on your system. You can download it from python.org.

Save the code in a file named calculator.py.

run it.
